LifeOS (Notion + Drive)

LifeOS integra organização tipo Notion com armazenamento no Google Drive — uma interface para organizar, buscar e abrir arquivos do seu Drive enquanto mantém os dados no seu dispositivo.

Quem mantém

Este projeto é mantido por Dalek2023 — um projeto pessoal/experimental. Qualquer dúvida ou solicitação de remoção de dados, entre em contato pela página de Contato.

Como funciona

  1. Você conecta sua conta Google Drive via OAuth.
  2. O app lê metadados dos arquivos (nome, id, tipo, tamanho) e, quando solicitado, lê o conteúdo de arquivos específicos para exibição ou indexação.
  3. Os dados acessados são armazenados localmente no seu dispositivo (veja a Política de Privacidade para detalhes).
